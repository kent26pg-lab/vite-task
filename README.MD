Handleliste

En enkel handleliste-app laget med Vite og vanilla JavaScript.

🔗 Live demo: https://kent26pg-lab.github.io/vite-task/

Funksjoner


Legg til varer i listen
Slett varer (med bekreftelse)
Varene lagres i localStorage, så listen huskes selv om du laster siden på nytt


Teknologi
Vite – byggeverktøy og dev-server
Vanilla JavaScript (ingen rammeverk)
uuid – genererer unik id for hver vare i listen
